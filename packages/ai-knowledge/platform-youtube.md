# Codes Video YouTube (Shorts + Long Format)

## Caracteristiques de la plateforme

- **Audience** : La plus diverse, toutes demographics, intent-driven (recherche)
- **Ton attendu** : Educatif, approfondi, storytelling structure
- **Duree optimale Shorts** : 30-60 secondes
- **Duree optimale long format** : 8-15 minutes (pour le personal branding)
- **Format Shorts** : 9:16 (vertical)
- **Format long** : 16:9 (horizontal)
- **Frequence ideale** : 2-3 Shorts/semaine + 1 video longue/semaine
- **Meilleur moment** : Vendredi-dimanche, 14h-17h

## Structure YouTube Shorts Performante

### Hook (0-3s)
- YouTube Shorts tolere un hook legerement plus long que TikTok
- Promesse claire : "En 60 secondes, tu vas apprendre a [resultat]"
- Curiosity gap : "Il y a 1 raison pour laquelle [probleme] et personne n'en parle"
- Preuve d'autorite rapide : "Apres 500 clients, voici le pattern que je vois..."

### Valeur pedagogique (3-45s)
- YouTube valorise le contenu EDUCATIF par-dessus tout
- Structure : Probleme → Explication → Solution → Exemple
- Chaque point doit etre completement comprehensible meme sans les autres
- Utiliser des analogies pour simplifier les concepts complexes
- B-rolls et schemas/diagrammes pour illustrer

### CTA + Boucle (45-60s)
- "Abonne-toi pour un conseil comme ca chaque jour"
- "Regarde cette video pour aller plus loin" (lien vers le long format)
- Technique de la boucle : terminer la video de maniere a ce que le replay soit naturel

## Sous-titres YouTube

### Pour les Shorts
- **Style** : Clean, lisible, pas trop stylise
- **Font** : Sans-serif standard, taille moyenne
- **Couleur** : Blanc sur fond sombre ou avec outline noire
- **Position** : Centre ou tiers inferieur
- **Animation** : Sobre — apparition mot par mot ou phrase par phrase

### Pour le long format
- **Sous-titres integres** via le systeme YouTube (SRT/VTT)
- OU sous-titres burns-in pour les highlights/citations
- **Keywords visuels** a l'ecran pour les points cles
- **Chapitres** (timestamps dans la description) sont essentiels

## Formats qui performent sur YouTube

### Shorts
1. **Le micro-tuto** : "Comment faire [X] en 60 secondes"
2. **Le fait surprenant** : "Tu savais que [stat] ?"
3. **Le debunk rapide** : "Non, [croyance populaire] est faux. Voici pourquoi."
4. **Le storytelling compress** : Une anecdote en 45s avec chute

### Long format (pour le personal branding avance)
1. **Le guide complet** : "Tout ce que tu dois savoir sur [sujet]"
2. **Le case study** : "Comment j'ai [resultat] — process complet"
3. **L'interview** : Conversation avec un expert du domaine
4. **Le behind-the-scenes** : "Une journee dans ma vie de [metier]"

## Ce qui ne marche PAS sur YouTube

- Shorts sans valeur educative (YouTube n'est pas TikTok)
- Clickbait sans delivrer la promesse (YouTube penalise via le dislike ratio)
- Mauvaise qualite audio (YouTube est tres sensible a l'audio)
- Pas de structure claire (YouTube valorise la retention curve)
- Contenus trop promotionnels sans valeur
- Ne pas optimiser le titre et la miniature (crucial pour le CTR)

## Algorithme YouTube (2025-2026)

### Shorts
- **Swipe-away rate** (% de gens qui swipent avant la fin) est la metrique #1
- Le **replay rate** est extremement puissant
- YouTube Shorts sont montres sur la page d'accueil ET dans l'onglet Shorts
- Les Shorts peuvent driver des abonnements vers la chaine (pipeline vers le long format)

### Long format
- **CTR (Click-Through Rate)** de la miniature determine la portee initiale
- **Average View Duration** determine si YouTube continue a recommander
- **Session time** (est-ce que le spectateur reste sur YouTube apres ta video) est le signal ultime
- Les **chapitres** ameliorent l'engagement et le SEO
- YouTube est un MOTEUR DE RECHERCHE — le SEO du titre/description/tags compte enormement
