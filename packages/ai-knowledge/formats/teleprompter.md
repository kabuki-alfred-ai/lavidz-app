# Format : Teleprompter

## Principe
L'IA genere un script structure que l'utilisateur lit/suit face camera. Le script est affiche en overlay transparent sur l'ecran d'enregistrement. Ideal pour les contenus qui necessitent precision et structure.

## Code viral exploite
- Storytelling maitrise (hook, structure, CTA integres)
- Delivery fluide et professionnelle
- Controle total sur le message

## Structure d'enregistrement

### Le script IA
Le script est decoupe en sections avec des marqueurs visuels :

```
[HOOK — 15s]
"Le CV est mort. Et si tu recrutes encore en le lisant ligne par ligne,
tu perds tes meilleurs candidats."

[CONTEXTE — 15s]
"Je dis ca parce qu'en 10 ans de conseil en recrutement,
j'ai vu des centaines d'entreprises rejeter des candidats exceptionnels
pour des raisons absurdes."

[VALEUR — 30-45s]
"Voici ce que je regarde a la place du CV :
• Premier point — la capacite a resoudre un probleme reel
• Deuxieme point — la coherence du parcours, pas les lignes
• Troisieme point — l'energie en entretien, pas les diplomes"

[CTA — 10s]
"Si tu veux ma methode complete, commente METHODE
et je te l'envoie."
```

### Guidage a l'ecran
- Texte defilant a vitesse ajustable (lent / normal / rapide)
- Section active surlignee, reste en opacite reduite
- Timer par section pour guider le rythme
- Possibilite de mettre en pause et reprendre
- Le texte est en bullet points, PAS un script mot-a-mot (pour garder le naturel)

### Duree cible
- Script total : 60-90 secondes de parole
- 3-4 sections maximum
- Chaque section : 15-30 secondes

## Regles de generation IA

L'IA doit generer des scripts qui :
- Sont en **bullet points guides**, pas en texte lu mot-a-mot
- Incluent un hook fort dans les 15 premiers mots
- Ont une structure narrative claire (probleme → solution → preuve → CTA)
- Utilisent le vocabulaire naturel du createur (analyse des sessions precedentes)
- Incluent des marqueurs de rythme : [PAUSE], [INSISTER], [SOURIRE]
- Sont adaptes a la plateforme cible (ton, duree, style)

## Montage type
- Zoom dynamique sur les moments d'insistance
- B-roll sur les points de liste (pendant que le createur parle)
- Texte a l'ecran qui renforce les mots-cles
- Transition smooth entre les sections
- Le teleprompter n'apparait PAS dans la video finale

## Regles de personnalisation (memoire IA)

### Le script n'est PAS generique
Chaque bullet point DOIT contenir un detail du createur :
- MAUVAIS : "Parlez de votre experience en recrutement"
- BON : "En 10 ans, j'ai recrute [X] personnes. L'erreur que je voyais tout le temps, c'est..."

### Le hook DOIT choquer
Utiliser un chiffre ou fait personnel tire de la memoire :
- MAUVAIS : "Le recrutement est broken"
- BON : "J'ai rejete 200 CV le mois dernier. Aucun ne meritait d'etre lu."

### Les marqueurs de performance
Inserer des indications emotionnelles basees sur le style du createur :
- [PAUSE — laisse le chiffre atterrir]
- [TON DIRECT — comme quand tu parles a un pote]
- [INSISTE — c'est ta conviction la plus forte]
- [RACONTE — insere ton anecdote ici, ne lis pas]

### Vocabulaire du createur
Le script DOIT utiliser les expressions et tournures favorites du createur.
Puiser dans la memoire pour reproduire SA voix, pas une voix corporate generique.

### Garde-fou anti-generique
Chaque section du script DOIT contenir au moins 1 element parmi :
- Un chiffre personnel
- Une anecdote vecue
- Une opinion tranchee
- Une expression favorite du createur
Si ce n'est pas le cas → reformuler avec un element de la memoire.

## Declinaison multi-plateforme
- **LinkedIn** : Script complet (60-90s), ton expert
- **TikTok** : Hook + 1 point valeur seulement (30-45s), ton energique
- **Instagram** : Script complet avec B-rolls esthetiques (45-60s)
- **YouTube Short** : Script complet, ton pedagogique (45-60s)
