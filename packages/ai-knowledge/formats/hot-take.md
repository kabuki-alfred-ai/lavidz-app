# Format : Take Chaud

## Principe
L'utilisateur reagit a un sujet d'actualite, une tendance, ou une opinion populaire de son secteur. Format court, opinione, spontane. L'IA propose le sujet et le cadrage, l'utilisateur reagit naturellement.

## Code viral exploite
- Reactivite (timing avec l'actualite)
- Opinion tranchee (polarise = engagement)
- Format court et punchy (algo-friendly)
- Debat en commentaires

## Structure d'enregistrement

### Briefing IA (avant l'enregistrement)
L'IA presente :
- Le sujet / la tendance identifiee
- 2-3 angles possibles de reaction
- Un hook suggere pour demarrer
- La position attendue (pour/contre/nuancee)

### Guide a l'ecran
```
[HOOK — 5s]
Affirmation choc ou question provocante sur le sujet

[TON AVIS — 30-40s]
• Ton opinion en 1 phrase
• Pourquoi tu penses ca (1 argument fort)
• Un exemple ou une preuve concrete

[OUVERTURE — 10s]
Question au public pour generer le debat
```

### Duree cible
- 30-60 secondes maximum
- Le plus court = le mieux pour ce format
- L'energie doit etre haute du debut a la fin

## Regles de generation IA

L'IA doit :
- Identifier des sujets trending dans le secteur du createur
- Proposer des angles qui sont en coherence avec les opinions passees du createur
- Formuler des hooks provocants mais pas diffamatoires
- Suggerer une prise de position claire (pas de "ca depend")
- Inclure un element de preuve (stat, anecdote, fait) pour credibiliser
- Proposer une question ouverte pour le CTA qui genere du debat

## Montage type
- Montage ultra-rapide (couper TOUS les blancs)
- Zoom ins sur les moments d'emphase
- Texte a l'ecran pour les affirmations cles
- Pas de B-roll (le format est 100% talking head, authenticite max)
- Sous-titres gros et animes (style TikTok/Reels)

## Regles de personnalisation (memoire IA)

### Selection du sujet
Tu DOIS croiser :
1. Les tendances actuelles du secteur du createur
2. Les opinions tranchees stockees en memoire
3. Trouver l'intersection : un sujet trending + une opinion forte du createur

EXEMPLE :
- Trending : "L'IA remplace les consultants"
- Memoire : "Le createur pense que l'IA augmente les consultants"
- → Hot Take : "Tout le monde dit que l'IA va tuer le consulting. J'ai integre l'IA dans mes process et mon CA a fait +40%."

### Hook personnalise
Le hook DOIT TOUJOURS contenir un element personnel :
- MAUVAIS : "L'IA va-t-elle remplacer les consultants ?"
- BON : "J'ai integre ChatGPT dans mon process de consulting. Mon CA a augmente de 40%. Et pourtant..."

### Energie calibree
Adapter le ton au style de communication du createur :
- Si le createur est direct/cash → "C'est du bullshit. Point."
- Si le createur est pedagogue → "Laissez-moi vous expliquer pourquoi cette idee est fausse."
- Si le createur est storyteller → "Il y a 6 mois, j'y croyais aussi. Et puis j'ai teste."

### Garde-fou anti-generique
Le hot take DOIT contenir au moins UN element parmi :
- Un chiffre personnel du createur
- Une anecdote vecue
- Une reference a une experience reelle
Si ce n'est pas le cas → reformuler avec un element de la memoire.

## Declinaison multi-plateforme
- **TikTok** : Format natif ideal (30-45s, max energie)
- **Instagram Reels** : Meme contenu, sous-titres plus soignes (30-45s)
- **LinkedIn** : Rallonger legerement avec plus de contexte (45-60s)
- **YouTube Short** : Meme contenu que TikTok (30-45s)
