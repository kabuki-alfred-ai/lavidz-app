# Format : Conseil du Jour

## Principe
Un conseil actionnable ultra-court. L'utilisateur delivre UNE idee, UN tip, UNE methode en 30-45 secondes. C'est le format le plus simple a produire et le plus "saveable" (les gens enregistrent pour plus tard).

## Code viral exploite
- Valeur immediate (le spectateur apprend quelque chose en 30s)
- Save-friendly (les saves sont le signal #1 sur Instagram et LinkedIn)
- Format reproductible a l'infini (1 conseil = 1 video)
- Facile a consommer = haut taux de completion

## Structure d'enregistrement

### Guide a l'ecran
```
[HOOK — 5s]
"Le conseil : [titre court et accrocheur]"
Exemple : "Arrete de faire tes calls debout."

[EXPLICATION — 20-30s]
• Pourquoi ce conseil marche
• Comment l'appliquer concretement
• Un exemple rapide ou une preuve

[RECAP — 5s]
"En resume : [reformuler le conseil en 1 phrase]"
```

### Duree cible
- 30-45 secondes MAXIMUM
- C'est le format le plus court — la concision est la regle
- Si le conseil necessite plus de 45s, c'est un format Teleprompter

## Regles de generation IA

L'IA doit :
- Identifier des micro-conseils dans l'expertise du createur
- Formuler le conseil en 1 phrase-titre accrocheuse
- Preparer 2-3 bullet points d'explication (pas plus)
- S'assurer que le conseil est ACTIONNABLE (pas theorique)
- Le conseil doit pouvoir etre applique le jour meme par le spectateur
- Varier les angles : methode, erreur a eviter, outil, habitude, mindset
- Ne pas repeter les conseils deja donnes dans les sessions precedentes

## Montage type
- Montage ultra-sec (zero temps mort)
- Texte titre a l'ecran pendant toute la video
- Transition rapide entre les points
- Pas de musique ou tres discrete
- Sous-titres gros et clairs
- Eventuellement un B-roll de 3s pour illustrer le conseil

## Regles de personnalisation (memoire IA)

### Selection du conseil
Le conseil DOIT venir d'une routine ou methode REELLE du createur stockee en memoire.
JAMAIS un conseil theorique ou generique.

Memoire → Tip :
- "Fait un 1-to-1 de 15 min chaque lundi" → "Le lundi matin, je fais un truc qui a transforme mon equipe..."
- "Utilise Notion pour tout organiser" → "Un seul outil a divise mon temps admin par 3..."

### Le hook du conseil
Formuler TOUJOURS avec un RESULTAT mesurable :
- MAUVAIS : "Voici un conseil de productivite"
- BON : "Ce truc me fait gagner 3h par semaine"
- MAUVAIS : "Voici comment je gere mes clients"
- BON : "Depuis que je fais ca, zero client m'a ghoste"

### Vocabulaire du createur
Utiliser SES mots, pas des mots corporate :
- Si le createur dit "je kiffe" → le script dit "je kiffe"
- Si le createur est formel → le script reste formel
- Puiser dans les expressions favorites stockees en memoire

### Garde-fou anti-generique
Le conseil DOIT etre quelque chose que le createur FAIT REELLEMENT.
Verifier dans la memoire : routines, outils, methodes mentionnees.
Si aucune routine n'est en memoire → poser la question en mode exploration :
"C'est quoi LE truc que tu fais chaque jour et qui fait la difference ?"

## Declinaison multi-plateforme
- **TikTok** : Format natif parfait (30s, direct, energique)
- **Instagram** : Meme contenu, sous-titres brandes (30-45s)
- **LinkedIn** : Ajouter 15s de contexte professionnel (45s)
- **YouTube Short** : Meme contenu, hook legerement plus pedagogue (30-45s)
